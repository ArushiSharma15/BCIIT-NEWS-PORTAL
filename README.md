# BCIIT-NEWS-PORTAL
The BCIIT News Portal is a web-based application developed using Python and Django framework to provide a centralized platform for sharing and accessing college-related information. The system is designed to display updates related to students, faculty, and events in an organized and user-friendly manner.

The application includes multiple modules such as Student Information (clubs and activities), Faculty Details, Events Management, and an Enquiry System. Each module allows users to view relevant information along with images, enhancing engagement and usability. The enquiry feature enables users to submit queries, which are managed through the Django backend.

The project utilizes Django’s MVC architecture, integrating frontend technologies like HTML, CSS, and Bootstrap for responsive design and improved user experience. The backend handles dynamic content rendering, database management, and form processing. Images and data are managed efficiently using Django models and the admin panel.

Key functionalities include dynamic content display, image integration, clickable navigation, form handling, and database-driven information management. The project demonstrates practical knowledge of full-stack web development, including routing, template rendering, static file handling, and CRUD operations.

Overall, the BCIIT News Portal showcases the ability to build a scalable and interactive web application, combining both frontend design and backend logic effectively.
